Skin Disease Detection using CNN

Python TensorFlow Deep Learning EfficientNet

Project Overview

This project aims to develop an automated CNN-based system that can accurately diagnose various skin diseases using image inputs. The system leverages EfficientNetB2, data augmentation techniques, and transfer learning to classify skin lesions into 10 distinct categories with over 83% accuracy.

The Challenge

Skin diseases contribute significantly to global disease burden. Accurate diagnosis is critical, yet traditional methods are subjective and slow. Existing ML-based systems are often limited by poor accuracy and usability. Our system seeks to overcome these limitations with a CNN-based approach.

Our Approach

  • Preprocessed image data with resizing, normalization, and cleaning
  • Augmented data using rotation, flipping, brightness adjustment
  • Used EfficientNetB2 model with fine-tuned hyperparameters
  • Trained on a balanced dataset of 10 skin disease classes
  • Applied dropout and early stopping to prevent overfitting
